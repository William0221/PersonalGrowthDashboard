# 📱 把成長儀表板裝到手機

你的儀表板已經是完整的 PWA（可安裝、可離線）。手機要用，只需兩步：**①放上網拿網址 → ②手機加入主畫面**。

---

## 第一步：免費上線（擇一即可）

### 方法 A — Netlify Drop（最簡單，免註冊也能先試）

1. 電腦瀏覽器打開：https://app.netlify.com/drop
2. 把整個 `PersonalGrowthDashboard` 資料夾**直接拖進**網頁中央的虛線框。
3. 等幾秒，它會給你一個網址，例如 `https://隨機名稱.netlify.app`。
4. 這個網址就能在手機打開了（建議註冊免費帳號，網址才會永久保留）。
https://fabulous-taffy-9cf525.netlify.app
> 注意：要拖「資料夾」，讓 `index.html` 在最外層。

### 方法 B — GitHub Pages（適合想長期維護的人）

1. 註冊 GitHub 帳號，建立一個新的 repository。
2. 把資料夾內所有檔案上傳上去。
3. 進 repo 的 **Settings → Pages**，Source 選 `main` 分支、根目錄，儲存。
4. 約一分鐘後會得到網址：`https://你的帳號.github.io/repo名稱/`。

---

## 第二步：手機「加入主畫面」

開啟上面拿到的網址後：

**iPhone（Safari）**
1. 用 **Safari** 打開網址（Chrome 不支援安裝）。
2. 點下方「分享」鈕（方框加箭頭）。
3. 選「**加入主畫面**」→ 加入。
4. 主畫面就會出現火箭圖示，點開即全螢幕，像 App 一樣。

**Android（Chrome）**
1. 用 **Chrome** 打開網址。
2. 點右上角「⋮」選單。
3. 選「**安裝應用程式**」或「加到主畫面」。
4. 完成後在抽屜/主畫面出現圖示。

## 常見問題

- **沒網路能用嗎？** 可以。第一次連網開啟後，Service Worker 會把介面快取起來，之後離線也能開、能做 EQ／技術題。只有「技術新聞」需要連網才能更新。
- **手機和電腦進度會同步嗎？** 不會。進度存在各自裝置本機，是分開的。若需要跨裝置同步，要另外做雲端儲存（可再找我做）。
- **改了內容怎麼更新？** 重新上傳檔案，並把 `sw.js` 裡的 `CACHE_VERSION` 數字 +1（例如 `pgd-v1` → `pgd-v2`），使用者下次開啟就會拿到新版。
